# Tailwind JIT Starter React

나중에 파일구조 업데이트 할 예정!

## Setup

1. Clone repo

```bash
git clone https://github.com/saviomartin/tailwind-jit-starter-react.git
```

2. Install Packages

```bash
npm i
```

3. Watch Css

```bash
npm run watch:css
```

4. Start Server

```bash
npm start
```
